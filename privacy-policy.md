# プライバシーポリシー — CLASS Calendar LETUS++

最終更新日: 2026年5月25日

## 概要

CLASS Calendar LETUS++ は、東京理科大学の CLASS ポータルから時間割を取得し、Google カレンダーに登録する Chrome 拡張機能です。

## 取り扱うデータについて

本拡張機能は、開発者が運営する外部サーバーへ個人データを送信・収集・保存することは一切ありません。

### 端末内で扱うデータ

| データ | 保存場所 | 目的 |
|---|---|---|
| 学期の開始日・終了日 | Chrome ローカルストレージ（端末内のみ） | 次回起動時の入力省略 |
| 時間割データ（授業名・教室・時限） | メモリのみ（ポップアップを閉じると消去） | ポップアップへの表示とカレンダー登録 |
| Google OAuth アクセストークン | メモリのみ（ポップアップを閉じると消去） | Google カレンダー API の呼び出し |

## 通信先

本拡張機能が通信する外部サービスは以下の2つのみです。

- **CLASS ポータル**（`class.admin.tus.ac.jp`）— 時間割の取得
- **Google カレンダー API**（`googleapis.com`）— カレンダーへのイベント登録

その他の第三者サービスへのデータ送信は行いません。

## Google アカウントへのアクセスについて

Google ログイン時に要求する権限は `calendar.events`（イベントの作成・読み取り・削除）のみです。Google の認証情報は本拡張機能に保存されません。

この権限は、登録した予定の削除機能（履修変更時などに使用）のためにも使用されます。削除は本拡張機能で登録した予定のみを対象とし、それ以外の予定には一切アクセスしません。
