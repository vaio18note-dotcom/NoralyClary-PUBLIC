# CLASS Calendar LETUS++

東京理科大学の CLASS ポータルから時間割を取得し、Google カレンダーに自動登録する Chrome 拡張機能です。

---

## 対象

- 東京理科大学の学生
- Google Chrome を使用している方

---

## インストール方法

### Chrome Web Store からインストール（準備中）

### 手動インストール（開発者モード）

1. このリポジトリの `classcalendar` フォルダをダウンロードする
2. Chrome で `chrome://extensions/` を開く
3. 右上の「デベロッパーモード」をオンにする
4. 「パッケージ化されていない拡張機能を読み込む」をクリック
5. ダウンロードした `classcalendar` フォルダを選択する

---

## 使い方

### 準備

1. Chrome で **CLASS ポータル** にログインし、学生時間割表を表示する
2. 画面右上の拡張機能アイコンをクリックしてポップアップを開く
3. **学期開始日** と **学期終了日** を入力する

---

### 方法A：Google カレンダーに直接登録する

1. 「🔑 Google ログイン」ボタンをクリックし、Google アカウントでログインする
2. 登録したい授業の「📅」ボタン、または「📅 月〜金を一括登録」で登録する

### 方法B：ICS ファイルをダウンロードして取り込む（ログイン不要）

1. 「💾」ボタン、または「💾 月〜金を一括 DL」でファイルをダウンロードする
2. [Google カレンダー](https://calendar.google.com/) を開く
3. 右上の歯車アイコン →「設定」をクリックする
4. 左メニューの「インポート / エクスポート」タブを開く
5. ダウンロードした `.ics` ファイルを選択して「インポート」をクリックする

---

### 履修変更時：登録した予定をまとめて削除する

1. Google ログイン後、学期開始日・終了日を設定する
2. 「🗑 登録済み予定を削除」ボタンをクリックする
3. 件数が表示されたら、5 秒以内にもう一度クリックして確定する

> この拡張機能で登録した予定のみが削除対象になります。手動で追加した予定は削除されません。

---

## 登録される予定の内容

| 項目 | 内容 |
| --- | --- |
| タイトル | 授業名 |
| 場所 | 教室名 |
| 繰り返し | 毎週（学期終了日まで） |
| 月〜金の授業 | 時限ごとの時間帯で登録 |
| 土曜の授業 | 終日予定で登録 |

### 時限と時間帯

| 時限 | 時間 |
| --- | --- |
| 1 限 | 8:50 〜 10:20 |
| 2 限 | 10:30 〜 12:00 |
| 3 限 | 13:00 〜 14:30 |
| 4 限 | 14:40 〜 16:10 |
| 5 限 | 16:20 〜 17:50 |
| 6 限 | 18:10 〜 19:40 |
| 7 限 | 19:50 〜 21:20 |
